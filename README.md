Bigendra Shrestha's Learning Journey
Welcome to my personal blog, where I document my coding journey! This repository hosts the source code for my website (https://potat0ka.github.io), showcasing my growth as a coder through internships, online courses, books, and real-life experiences. Built using the Fractal template by HTML5 UP, this site is a living portfolio of my learning adventures.
About Me
Hi, I’m Bigendra Shrestha, an aspiring coder from Nepal. I’m passionate about learning new technologies and applying them to solve real-world problems. This blog tracks my progress through:

Internship with Nobel Internship via EduQuest.
DataCamp Courses to build my Python skills.
Coding Books like Automate the Boring Stuff with Python.
Real-Life Experiences from personal projects.
A fun Tic-Tac-Toe game with a scoreboard, added on May 27, 2025!

Features

Interactive pages for Internship, DataCamp Courses, Coding Books, and Real-Life Experiences.
A timeline of my coding milestones.
A Tic-Tac-Toe game with high score tracking using local storage.
Responsive design for desktop and mobile views.
Links to my GitHub profile (@potat0ka).

Getting Started
Prerequisites

A web browser (e.g., Chrome, Firefox) to view the site.
Git installed to clone or contribute to the repository.

Installation

Clone the repository:git clone https://github.com/potat0ka/potat0ka.github.io.git


Navigate to the project directory:cd potat0ka.github.io


Open index.html in a web browser to explore the site locally.

Usage

Visit https://potat0ka.github.io to see the live site.
Explore the navigation links under "Explore My Journey" to learn about my journey.
Play the Tic-Tac-Toe game at the bottom of the homepage and track your scores!
